## Desk

- **Desktop** — Arch Linux (yes, I say it), RTX 3060, RGB synced to the wallpaper because a system should feel like one thing.
- **Raspberry Pi 4** — the homelab. Photo backups, media, home automation, all in Docker, reachable over Tailscale.

## Environment

- **Hyprland** — tiling Wayland compositor, themed with Material You colors generated from the current wallpaper. The desktop repaints itself when the sky changes.
- **Neovim** — where the code happens. Config is [public](https://github.com/Pedronane/nvim).
- **JetBrains Mono** — in the terminal, in the editor, and on this site. Consistency is a feature.

## Habits

- Notes in Obsidian, in my own words — if I can't rewrite it, I haven't learned it.
- Spaced repetition for Japanese and physics. The forgetting curve is undefeated; flashcards are damage control.
- Guitar every day. Some dependencies you don't containerize.
